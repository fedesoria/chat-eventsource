Chat with EventSource

Chat application created on Sinatra using EventSource

View project onGitHub

Hola

Esta es una aplicación de prueba para #hermosillorb para demostrar el uso de EventSource y como lo podemos usar con mucha facilidad. En este caso estamos utilizando Sinatra y Heroku para subirlo a la web lo mas rapido y gratis posible.

Para clonar la app solo tienes que seguir los siguientes comandos en tu consola:

$ git clone git://github.com/fedesoria/chat-eventsource.git
$ cd chat-eventsource
$ bundle
$ ruby chat.rb

Deploy a Heroku

Si quieres poner en internet tu propia versión, ya esta todo listo para subirlo a Heroku solo agrega la aplicación/repositorio:

$ heroku create nombre_de_nueva_aplicacion
$ git push heroku master
